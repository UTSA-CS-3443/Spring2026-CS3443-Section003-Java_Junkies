# Budget Wise 💸

Budget Wise is a JavaFX application designed to help users track their personal budget and monitor daily spending. This application provides a straightforward way to manage financial data, record expenses, and keep track of budget limits.

## 🚀 Features

* **Expense Tracking:** Easily log daily purchases and expenses.
* **Budget Management:** Set and monitor spending limits.
* **Financial Summaries:** View a breakdown of remaining funds and total spending.
* **Categorial Organization:** Identify and organize specific types of spending.

## 🛠️ Built With

* **Java** - Core application logic and object-oriented design.
* **JavaFX** - GUI and Graphical Interface.
